# Die Anti-Bürokratie-Checkliste 2026
## Sozialer Navigator - Dein Recht. Einfach.

Damit dein Antrag sofort bearbeitet wird, lege diese Unterlagen direkt bei:

### 1. Persönliches
[ ] Personalausweis oder Reisepass (Kopie/Foto)
[ ] Meldebescheinigung
[ ] Krankenversichertenkarte

### 2. Einkommen (letzte 3 Monate)
[ ] Lohnabrechnungen (Netto)
[ ] Rentenbescheide (falls zutreffend)
[ ] Kindergeld-Nachweise
[ ] Unterhaltszahlungen (die du bekommst)

### 3. Wohnen
[ ] Aktueller Mietvertrag (Kaltmiete + Nebenkosten getrennt)
[ ] Letzte Heizkostenabrechnung
[ ] Nachweis über Mietzahlungen (Kontoauszug)

### 4. Vermögen (Nur bei Bürgergeld relevant)
[ ] Kontoauszüge der letzten 3 Monate (lückenlos)
[ ] Sparbücher / PayPal-Status

---
**Tipp:** Streiche auf den Kontoauszügen private Ausgaben (z.B. Supermarkt, Hobbys). Einnahmen müssen lesbar bleiben!
